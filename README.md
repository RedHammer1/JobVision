# JobVision

Инструкция по установке и запуску проекта JobVision

Перед началом работы убедитесь, что на вашем компьютере установлены Node.js версии 16 или выше и PostgreSQL. Для проверки выполните в командной строке команды node -v и psql --version. Если программы не установлены, скачайте их с официальных сайтов.

Установите зависимости проекта. Запустите файл install.bat двойным кликом. Этот скрипт установит все необходимые пакеты для бэкенда и фронтенда. Дождитесь завершения процесса, в консоли появится сообщение об успешной установке.

Настройте базу данных. Создайте базу данных с именем messenger_db. Используйте следующие параметры для подключения: имя пользователя admin, пароль password, хост localhost, порт 5432. Вы можете создать базу через pgAdmin или выполнить SQL-запрос CREATE DATABASE messenger_db;. Затем выполните SQL-скрипт create_vac_db.sql, который находится в корневой папке проекта, для создания всех необходимых таблиц.

Заполните базу данных вакансиями. Перейдите в папку Scrapper и установите Python-зависимости командой pip install -r requirements.txt. Затем запустите парсер командой python main.py. Скрипт соберет вакансии с Avito и сохранит их в базу данных. Процесс может занять несколько минут в зависимости от количества страниц. Если по какой-то причине скрапинг не удался или вы не хотите ждать его завершения, вы можете восстановить готовый бэкап базы данных. Файл бэкапа называется backup.sql и находится в корневой папке проекта. Для восстановления выполните команду в терминале: psql -U admin -d messenger_db -f backup.sql. После этого база данных будет заполнена тестовыми вакансиями, пользователями и сообщениями, и вы сможете сразу приступить к использованию приложения.

Запустите приложение. Запустите файл start.bat. Откроются два командных окна: для бэкенда и для фронтенда. Дождитесь сообщений о готовности. Бэкенд запустится на порту 8080, фронтенд на порту 5173. После запуска откройте браузер и перейдите по адресу http://localhost:5173.

Работа с приложением. Зарегистрируйтесь как преподаватель или студент. Преподаватель может создавать тесты, управлять требованиями к вакансиям и отправлять рекомендации в чат. Студент может проходить тесты, общаться в чате и получать доступ к рекомендованным вакансиям после успешного прохождения тестов. Для управления требованиями к вакансии преподаватель нажимает кнопку Управление доступом в карточке вакансии внутри чата и выбирает необходимый тест.

При повторном запуске используйте только файл start.bat, установка зависимостей не требуется.
